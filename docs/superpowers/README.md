# Superpowers Notes

Esta pasta foi preservada como documentação de apoio e histórico de planejamento.

## Regra de uso neste projeto
- O workflow oficial do repositório é `BMad`.
- O estado oficial do projeto deve ser lido em `docs/_bmad-output/`.
- Os módulos, templates e referências do método ativo ficam em `docs/_bmad/`.
- O conteúdo desta pasta **não substitui** o acompanhamento oficial do sprint, stories ou prioridades.

## Quando consultar esta pasta
- Quando o usuário pedir explicitamente para reutilizar um plano do `Superpowers`
- Quando um plano daqui ajudar como referência de implementação
- Quando for útil recuperar contexto histórico sem alterar o workflow ativo

## O que não fazer
- Não assumir que um plano em `docs/superpowers/` muda o status oficial do projeto
- Não usar esta pasta como fonte primária para decidir a próxima story
- Não apagar ou sobrescrever estes arquivos só por o projeto voltar para `BMad`
